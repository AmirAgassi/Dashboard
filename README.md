## Pre-Requisites

- PNPM
- Node 18.x
- Internet
- VS Code (Can't code without this)

## Development

Setting up local development is very easy

Steps:

1. `git clone git@github.com:LaurierHawkHacks/Dashboard.git`
2. `cd Dashboard`
3. `pnpm install`
4. `pnpm dev`
