## Development

Before you begin development, please read our [CONTRIBUTING](/docs/CONTRIBUTING.md) file.

Setting up local development is very easy:

Steps:

1. `git clone git@github.com:LaurierHawkHacks/Dashboard.git`
2. `cd Dashboard`
3. `pnpm install`
4. `pnpm dev`
